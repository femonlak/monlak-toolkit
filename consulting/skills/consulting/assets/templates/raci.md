## RACI Legend

| Code | Role | Definition |
|------|------|------------|
| **R** | Responsible | Does the work to complete the task |
| **A** | Accountable | Has final decision authority; only ONE per task |
| **C** | Consulted | Provides input before work is done (two-way) |
| **I** | Informed | Kept updated on progress (one-way) |

## Validation Checklist

Use this checklist to validate the RACI matrix:

- [ ] Each row has exactly ONE "A" (Accountable)
- [ ] Each row has at least one "R" (Responsible)
- [ ] No person has too many "A"s (overloaded)
- [ ] No person has too many "R"s (overloaded)
- [ ] Key stakeholders are included as C or I
- [ ] No gaps in critical tasks
- [ ] All team members reviewed and agreed
