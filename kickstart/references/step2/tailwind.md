# Tailwind CSS + Shadcn Configuration

Setup for web (Next.js/Vite) with Shadcn/ui components.

## Installation

```bash
npm install -D tailwindcss postcss autoprefixer
npm install tailwindcss-animate class-variance-authority clsx tailwind-merge
npm install lucide-react
npx tailwindcss init -p
```

## Utility Functions (src/lib/utils.ts)

```typescript
import { type ClassValue, clsx } from 'clsx'
import { twMerge } from 'tailwind-merge'

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs))
}
```

## Shadcn Component Setup

To add Shadcn components later:

```bash
# Initialize Shadcn (run after this setup is complete)
npx shadcn@latest init

# Add specific components as needed
npx shadcn@latest add button
npx shadcn@latest add card
npx shadcn@latest add input
```

**Note**: Don't run `shadcn init` during Step 2. Just set up the base configuration. Components will be added during development.

## Components Directory Structure

```
src/components/
├── ui/              # Shadcn components go here

```
